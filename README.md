# SocialMedia App ✨

A professional, full-stack social media platform with a premium dark-themed UI, real-time-like interactions, and personalized feeds.

## 🚀 Features

- **Modern UI/UX**: Built with Tailwind CSS and Lucide icons for a sleek, premium feel.
- **Personalized Timeline**: See posts from users you follow or discover everyone's content in the "For You" feed.
- **Real-time Interactions**: Support for Liking and Commenting on posts with instant UI updates.
- **User Discovery**: Advanced search functionality to find and follow other users.
- **Profile Management**: Customizable bios and profile stats (followers/following).
- **Responsive Design**: Fully optimized for desktop and mobile views.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS, Lucide React, Axios.
- **Backend**: Node.js, Express.js, JWT Authentication.
- **Database**: MongoDB (Mongoose).

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Shalini0410/social-media-app.git
   cd social-media-app
   ```

2. **Backend Setup**:
   ```bash
   cd backend
   npm install
   # Create a .env file with:
   # MONGO_URI=your_mongodb_uri
   # JWT_SECRET=your_jwt_secret
   # PORT=5000
   npm run dev
   ```

3. **Frontend Setup**:
   ```bash
   cd ../frontend
   npm install
   npm start
   ```
